Validador XML - Análise de Contas

Anexar Arquivo XML

Selecione um arquivo XML para iniciar a validação.

Validação por TAG

Insira o nome de uma TAG e clique em "Validar TAG" para verificar se ela está presente ou ausente nas contas do arquivo XML.

Observe o padrão das letras maiúsculas e minúsculas.
Ex: dadosAutorizacao.

Comparando as contas do arquivo

Clique para listar as TAGS presentes nas contas do arquivo, e verificar em quais contas as TAGS estão presentes e em quais estão ausentes.

Lista de Beneficiários

Clique para exibir a lista de beneficiários presente no arquivo XML.
